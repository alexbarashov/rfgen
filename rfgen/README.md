# rfgen (skeleton)

Универсальный RF-генератор: заготовка проекта (PySide6 UI минимально запускается).

## Структура
См. каталоги `rfgen/core`, `rfgen/standards`, `rfgen/backends`, `rfgen/ui_qt`, `rfgen/cli`.

## Запуск UI (PySide6)
```bash
python -m rfgen.ui_qt.app
```

## Требования
- Python 3.9+
- PySide6, numpy
